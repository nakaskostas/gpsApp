


# Product Requirements Document (PRD)
## Web App: Route Planner with Hover Tooltips

### 1. Overview
This PRD outlines the requirement to add hover tooltips to two toggle switches in the existing route planner web application. The tooltips should match the style and behavior of the existing "Estimated Duration" tooltip.

### 2. Requirements

#### 2.1 Round Trip Toggle Tooltip
- **Element**: The toggle switch labeled "Κυκλική Διαδρομή" (Round Trip)
- **Trigger**: Mouse hover
- **Tooltip Text**: "Ανοίγοντάς το, η διαδρομή σας γυρίζει αυτόματα πίσω στο σημείο εκκίνησης"
- **Behavior**: Tooltip appears on hover and disappears on mouse leave

#### 2.2 Steep Uphill Toggle Tooltip
- **Element**: The toggle switch labeled "Απότομη ανάβαση" (Steep Uphill)
- **Trigger**: Mouse hover
- **Tooltip Text**: "Δείχνει με κόκκινο χρώμα τις απότομες ανηφόρες (>5°) στον χάρτη"
- **Behavior**: Tooltip appears on hover and disappears on mouse leave

#### 2.3 Styling Requirements
- **Appearance**: Must match existing "Estimated Duration" tooltip
- **Positioning**: Should appear above the toggle switches
- **Animation**: Fade in/out effect identical to existing tooltip
- **Duration**: Same transition timing as existing tooltip
- **Arrow**: Include small arrow pointing to the toggle element
- **Styling**: Same background color (#1f2937), text color (white), padding (8px 12px), border-radius (6px), font-size (14px), and max-width (300px)

#### 2.4 Technical Specifications
- **Implementation**: Use same CSS classes and JavaScript structure as existing tooltip
- **Z-index**: Should appear above all other UI elements
- **Responsive**: Must adjust position on window resize
- **Accessibility**: Should not interfere with toggle functionality

### 3. Implementation Notes
- Tooltips should be created dynamically when hover occurs
- Use existing tooltip system as reference for consistency
- Ensure tooltips are properly cleaned up on mouse leave
- No changes to existing toggle functionality

### 4. Acceptance Criteria
- [ ] Hovering over "Κυκλική Διαδρομή" shows correct tooltip text
- [ ] Hovering over "Απότομη ανάβαση" shows correct tooltip text
- [ ] Tooltips disappear when mouse leaves toggle area
- [ ] Tooltip styling matches existing "Estimated Duration" tooltip
- [ ] Toggle functionality remains fully operational
- [ ] Tooltips work on both desktop and mobile devices
- [ ] No performance degradation observed
