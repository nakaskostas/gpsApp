θέλω να φτιάξω ένα web App. Θελω να μου γραψεις ενα πλάνο του Project Requirement document για να μπορέσω να φτιάξω το τελικό 
 Project Requirement document ετσι ωστε να το δωσω έτοιμο σε ένα LLM και να μου κάνει όλο το web app. Σκοπός μου ειναι να μην

Παρακατω σου δίνω ενδυκτικά τον σκοπό του app. Μην δωσεις δημασία στο τεχνικό κομμάτι του development, αυτό θα το κάνω εγώ,
ουσιαστικά θέλω ένα Project Requirement document για να του δωσω έπειτα το τι θέλω να μου κάνει ακριβώς, για τωρα θελω 
απλά ένα πλάνο του Project Requirement document:


Σκοπός του Web App:
1. Να έχει μία αρχική σελίδα που να εμφανίζει τον χαρτη και να μπορεί ο χρήστης να βάζει pins στο χαρτη 
(τον χαρτη θα τον πάρω μέσα API από openstreetmaps και opensourceservice). O χρηστης θα μπορει να βαζει pin πάνω στον χαρτη. Το 1ο pin 
θα είναι η αρχική τοποθεσία. 'επειτα σε καθε pin θέλω να βελτιστοποιεί την διαδρομή από pin σε pin που δίνει ο χρήστης μόνο στο 
οδικό δίκτυο. 
2. Θέλω κάτω από τον χάρτη να μου βγάζει την υψομετρικη όλης της διαδρομής, όπως το elevation profile στο app του komoot, και να 
έχει διαφορετικό χρώμα στα σημεία με κλήση μεγαλύτερη τον 5 μοίρες.

!ΣΗΜΑΝΤΙΚΟ ότι το app θέλω να τρέχει live σε μια οθόνη και δε θα υπάρχει κάποια βάση δεδομένων από πίσω. Όλη η πληροφορία
απλα θα έρχεται μέσο του openstreetmaps API.
